  EcoSip - Smart Bottle Website

  Project Description

This repository contains the source code for the EcoSip website, a static 8-page web application designed for a fictional smart water bottle company. The project demonstrates proficiency in HTML5 and CSS3 without the use of external frameworks (like Bootstrap) or JavaScript libraries, adhering strictly to the academic guidelines provided for Capstone Project 1-A.

The website showcases the "EcoSip" product, featuring technical specifications, a companion app preview, customer reviews, and a responsive navigation system.

   Features

  * 8-Page Structure: A fully navigable site including Home, Product, Features, App, Reviews, FAQ, About Us, and Enquiry pages.
  * Sticky Navigation Bar: A fixed header that remains accessible while scrolling.
  * Responsive Design: Utilizes CSS Flexbox and percentage-based padding to adapt to different screen sizes.
  * CSS Gradients: Implements a modern `linear-gradient` background with a fixed attachment for a premium aesthetic.
  * Static Forms: Includes a styled Enquiry form and contact section.
  * Semantic HTML: Proper use of `<header>`, `<main>`, `<nav>`, `<section>`, and `<footer>` tags.

 File Structure

| File Name | Description |
| :--- | :--- |
| `index.html` | Home Page: Welcome banner and brand introduction. |
| `product.html` | Product Overview: Visuals and summary of the bottle variants. |
| `features.html` | Specs: Detailed technical specifications table (Battery, Material, etc.). |
| `app.html` | Companion App: Overview of the mobile app features and interface. |
| `reviews.html` | Testimonials: Customer feedback displayed in styled cards. |
| `faq.html` |Help: Frequently Asked Questions section. |
| `about.html` | About Us: Company story, mission, and developer profile. |
| `enquire.html` | Contact: A static HTML form for user inquiries. |
| `style.css` | Stylesheet: The single external CSS file controlling the design system. |
| `EcoSipLogo.png` | Assets: Logo image used in the header. |
| `EcoSipBottle.png`| Assets: Product image used on the Home and Product pages. |

 Technologies Used

  * HTML5: Structure and semantic markup.
  * CSS3: Styling, layout (Flexbox/Grid), and visual effects.
  * Git/GitHub: Version control and project submission.

 How to Run This Project

Since this is a Static Website, no backend server or installation is required.

1.  Clone the Repository:
    ```bash
    git clone https://github.com/[Your-Username]/EcoSip-Website.git
    ```
2.  Open the Project:
    Navigate to the project folder on your computer.
3.  Launch:
    Double-click `index.html` to open the website in your default web browser (Chrome, Firefox, Edge, etc.).
